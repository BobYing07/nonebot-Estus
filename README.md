### 实现
本项目可以在 **Linux/Windows** 下运行，

首先需要具备 `Python 3.9+` 的环境 （P.S. `3.7+` 应该也行)

然后安装 `go-cqhttp` +  `nonebot1`

更新日志请关注Update.md




### 功能
执行命令：`/menu`
